# 第1章：Node.js简介

在本章中，我们将开始一段探索Node.js迷人世界的旅程。我们将从理解Node.js是什么以及它如何革新Web开发领域开始。所以，请系好安全带，准备好深入Node.js编程的世界！

## 1.1 什么是Node.js？

Node.js是一个开源的服务器端JavaScript运行时环境，允许开发者构建可扩展且高性能的应用程序。与传统的Web开发框架不同，Node.js采用事件驱动的非阻塞I/O模型，使其在处理并发请求时异常高效，非常适合构建高并发的应用。

## 1.2 为什么选择Node.js？

选择Node.js作为Web开发项目的原因有几个非常有说服力的因素：

### 1.2.1 JavaScript无处不在

Node.js的一个重要优势是它使用JavaScript作为主要编程语言。JavaScript是一种广泛采用的语言，这意味着开发者可以使用相同的语言编写客户端和服务器端代码，从而简化开发过程并促进代码重用。

### 1.2.2 可扩展性和性能

Node.js基于Google开发的V8 JavaScript引擎构建，它将JavaScript代码直接编译为机器代码。这一编译过程使得Node.js能够构建出高性能的应用程序，处理大量并发连接时不会消耗过多系统资源。

### 1.2.3 非阻塞I/O

Node.js采用事件驱动的非阻塞I/O模型，这意味着它可以高效地同时处理多个请求，而不会阻塞其他操作的执行。这种架构使得Node.js特别适合构建实时应用程序，例如聊天服务器或流媒体平台。

### 1.2.4 丰富的包生态系统

Node.js 拥有一个充满活力且广泛的包和库生态系统，称为 Node 包管理器（NPM）。NPM 提供了大量预构建的模块，开发者可以轻松地将其集成到自己的项目中，从而节省时间和精力。

## 1.3 开始使用 Node.js

在深入了解 Node.js 开发之前，你需要设置好开发环境。请按照以下步骤开始：

### 1.3.1 安装 Node.js

要安装 Node.js，请访问官方 Node.js 网站（https://nodejs.org）并下载适合你操作系统的安装程序。下载后，运行安装程序并按照安装向导的指示进行操作。

### 1.3.2 验证安装

安装 Node.js 后，打开命令提示符或终端，并输入以下命令来检查 Node.js 是否正确安装：

```js

node -v

```

如果 Node.js 安装正确，命令将显示已安装的 Node.js 版本。

### 1.3.3 使用 Node.js REPL

Node.js 配备了内置的交互式读取-求值-输出循环（REPL），允许你实时实验 JavaScript 代码并执行它。要启动 Node.js REPL，请打开命令提示符或终端，并输入以下命令：

```js

node

```

你将看到 Node.js 提示符（">"），表示你现在处于 REPL 环境中。你可以开始输入 JavaScript 代码，并立即看到结果。
